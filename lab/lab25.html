The package mice implements a simple version of multiple imputation. We can create 4 different imputations by calling: 

imp = mice(data, m = 4)

We can specify the method of imputation for numerical and categorical variables using the parameter method. There are many options, for example, sampling from the non-missing values for each variable: 

imp = mice(data, m = 4, method = "sample")

Or predicting using linear regression without error (m = 1 yields single imputation): 

imp = mice(data, m = 1, method = "norm.predict")

The alternative package mi detects a class for each variable and uses sophisticated default regression methods for each class.

Low-rank matrix completion

The package softImpute implements an algorithm that completes a data matrix while restricting the rank of the completion. The function softImpute allows you to specify a maximum rank: 

softImp = softImpute(as.matrix(data), rank.max = 60)
data.completed = softImp$u %*% diag(softImp$d) %*% t(softImp$v)

We can examine the MSE of the completion on the non-missing entries: 

mean((data.completed[!is.na(data)] - data[!is.na(data)])^2)
## [1] 0.07941561

This averaged deviation will increase as we reduce the rank. For example: 

softImp = softImpute(as.matrix(data), rank.max = 30)
data.completed = softImp$u %*% diag(softImp$d) %*% t(softImp$v)
mean((data.completed[!is.na(data)] - data[!is.na(data)])^2)
## [1] 13.40608

In practice, we could use rank.max as a tuning parameter and select it through cross-validation. The function softImpute also takes a parameter lambda, which determines a penalty on the nuclear norm of the completed matrix. You can also use this as a tuning parameter.
